функция main расположена в файле BezierСurves.cpp

Програма будет открывать файл с именем inputData.txt, 
считывать входные данные из файла и записывать рассчитанную кривую Безье 
в файл с именем outputData.txt

Формат входных даных:
Сначала вводятся координаты точек для построения плоскости
x0 - координата х первой точки, y0 - координата y первой точки
x0;y0;z0 
x1;y1;z1
x2;y2;z2
После точек задающих плоскость, идут опорные точки кривой Безье в том же формате
x3;y3;z3 
x4;y4;z4
...
xn;yn;zn 

Точка под номером n - последняя точка из опорных точек кривой
